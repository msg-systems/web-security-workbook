# OWASP Top 10 2021

The Open Web Application Security Project (OWASP) is a foundation whose goal is to improve the security of software. Every two to three years, they release a new list of security risks which has high reputation and also is often quoted in scientific literature.

For more information regarding the list please see OWASP (opens new window).

The current top ten risks are the following:

A01:2021

  • Broken Access Control

A02:2021

  • Cryptographic Failures

A03:2021

  • General Information
  • SQL & NoSQL Injection
  • Cross-Site Scripting (XSS)

A04:2021

  • Insecure Design

A05:2021

  • General Information
  • Error Handling
  • HTTP Headers
  • XML External Entities (XXE)

A06:2021

  • Vulnerable and Outdated Components

A07:2021

  • General Information
  • JSON Web Token (JWT)
  • Passwort Hashing
  • Password Policies
  • Session Cookies

A08:2021

  • General Information
  • Insecure Deserialization

A09:2021

  • Security Logging and Monitoring Failures

A10:2021

  • Server-Side Request Forgery (SSRF)
